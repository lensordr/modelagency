## ğŸ¨ Customization

### Adding Screenshots
1. Save your screenshots in the `images/` folder
2. Replace the placeholder divs in `index.html` with:
```html
<img src="images/your-screenshot.jpg" alt="Description">
```

### Adding Demo Video
1. Save your video in the `videos/` folder
2. Replace the video placeholder with:
```html
<video controls>
    <source src="videos/your-demo.mp4" type="video/mp4">
</video>
```

### Updating Content
- Edit text content directly in `index.html`
- Modify colors and styling in `css/style.css`
- Add interactive features in `js/script.js`
